📋 Configuración Google Sheets

1.Ve a script.google.com → Nuevo proyecto
2.Borra el código que hay y pega el contenido del archivo google_apps_script.js
3.Pulsa Implementar → Nueva implementación → Aplicación web
4.En "Quién puede acceder" elige Cualquier usuario → Implementar
5.Copia la URL que te da y pégala aquí abajo
